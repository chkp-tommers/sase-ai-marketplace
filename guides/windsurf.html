Windsurf Setup Guide

Install and configure AI tools in Windsurf development environment

Overview

Windsurf is an AI-powered development environment that supports MCP (Model Context Protocol) servers, custom prompts, chat modes, and development rules. This guide will walk you through setting up various AI tools to enhance your development workflow.

Installation

Windsurf is available for Mac, Windows, and Linux. Download and install the latest version from the official site:

  • Getting Started Guide

Minimum requirements: OS X Yosemite (Mac), Windows 10, Ubuntu 20.04+, glibc >= 2.28.

After installation, follow the onboarding flow to set up your environment, import settings, and sign in or create a Windsurf account.

Key Features

  • Cascade: AI-powered assistant for code, chat, and workflow automation.
  • Tab: Context-aware code completion and navigation engine.
  • Command: In-line code generation, edits, and refactoring.
  • MCP: Model Context Protocol servers for extending AI capabilities.
  • Terminal: Integrated terminal with context awareness and advanced features.
  • Recommended Plugins: Enhance Windsurf with popular extensions from the Open VSX marketplace.
  • Previews: Live previews for web development and more.
  • App Deploys: One-click application deployment from within the IDE.
  • Web Search: Search the web directly from Windsurf.
  • Memories: Persistent context and rules for your projects.
  • Workflows: Automate repetitive tasks and project flows.
  • Planning Mode: Structured, multi-step AI planning for complex tasks.

MCP Server Setup

1. Create MCP Configuration

Create or edit your MCP configuration file in the project root: 

{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["@playwright/mcp"],
      "env": {}
    },
    "filesystem": {
      "command": "npx",
      "args": ["@filesystem/mcp"],
      "env": {
        "ALLOWED_PATHS": "/workspace"
      }
    },
    "git": {
      "command": "git-mcp",
      "args": [],
      "env": {}
    }
  }
}

2. Install MCP Dependencies

Install the required MCP servers: 

# Install Playwright MCP
npm install -g @playwright/mcp

# Install Filesystem MCP
npm install -g @filesystem/mcp

# Install Git MCP
npm install -g git-mcp

3. Restart Windsurf

Restart Windsurf to load the new MCP configuration. The servers will be available in your AI chat.

Custom Prompts

Adding Custom Prompts

Create a .windsurf/prompts.md file: 

# Code Review Assistant

You are an expert code reviewer with deep knowledge of security, performance, and best practices.

## Your Role
- Review code for bugs, security vulnerabilities, and performance issues
- Suggest improvements and alternatives
- Check for code style and convention adherence

## Review Checklist
1. **Security**: SQL injection, XSS, authentication, authorization
2. **Performance**: Algorithms, database queries, memory usage
3. **Maintainability**: Code clarity, documentation, modularity

---

# API Design Expert

You are an expert in REST API design, following industry best practices.

## Your Expertise
- RESTful API design patterns
- OpenAPI/Swagger documentation
- Authentication and authorization
- Error handling and status codes

Using Prompts

  1. Open the AI chat in Windsurf
  2. Type @prompt-name to use a specific prompt
  3. Or copy the prompt content and paste it in your conversation

Chat Modes

Setting Up Chat Modes

Create specialized AI assistants by configuring chat modes in .windsurf/chat-modes.json: 

{
  "modes": {
    "api-architect": {
      "name": "API Architect",
      "description": "Expert in RESTful API design and documentation",
      "systemPrompt": "You are an expert API architect specializing in RESTful API design, microservices architecture, and API documentation...",
      "temperature": 0.7,
      "maxTokens": 2048
    },
    "security-auditor": {
      "name": "Security Auditor",
      "description": "Security-focused code review and vulnerability assessment",
      "systemPrompt": "You are a cybersecurity expert specializing in application security...",
      "temperature": 0.3,
      "maxTokens": 2048
    }
  }
}

Development Rules

Creating Rules File

Add development rules in .windsurf/rules.md: 

# TypeScript Strict Configuration

## Compiler Options
- Use strict mode: `"strict": true`
- No implicit any: `"noImplicitAny": true`
- Strict null checks: `"strictNullChecks": true`

## Coding Standards
- Use explicit type annotations
- Handle null/undefined cases
- Prefer interfaces over types for objects

# React Best Practices

## Component Guidelines
- Use functional components with hooks
- Implement proper error boundaries
- Optimize with React.memo when needed

## Performance
- Code splitting with React.lazy
- Virtualization for large lists
- Proper key props for lists

Troubleshooting

Common Issues

MCP Server Not Found

Ensure the MCP package is installed globally and the configuration file is in the correct location.

Configuration Not Loading

Check JSON syntax and restart Windsurf after making changes.

Prompts Not Working

Verify the prompts file is properly formatted and saved in the correct directory.
